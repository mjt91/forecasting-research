# Forecasting Research

Time series forecasting research — papers, articles, notes. Built with the llm-wiki pattern.

Run by Hermes Agent weekly. Ingest sources: arXiv (q-fin.ST, stat.ML), Hacker News, practitioner blogs.

## Structure

```
SCHEMA.md           # Conventions and tag taxonomy
index.md            # Content catalog
log.md              # Action log (append-only)
raw/
  papers/           # Immutable source papers (arXiv etc.)
  articles/         # Immutable source articles
concepts/          # Concept/topic pages
models/             # Model architecture pages
datasets/          # Dataset pages
```

## Tags

- `forecasting` — general forecasting
- `time-series` — time series specific
- `transformer` — transformer-based methods
- `neural-net` — neural network architectures
- `statistical` — classical statistical methods
- `finance` — financial applications
- `production` — MLOps, deployment
- `arxiv` — sourced from arXiv
- `blog` — blog posts, articles
- `data` — datasets
